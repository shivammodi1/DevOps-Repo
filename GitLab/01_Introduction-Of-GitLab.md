# Introduction Of GitLab

- GitLab ek web-based DevOps platform hai.
- Yeh complete CI/CD toolchain provide karta hai (code se deployment tak).
- GitLab cloud-hosted aur self-managed, dono deployment support karta hai.

## Important Features:
•Cloud-hosted deployment:
- GitLab service provider ke server par host hota hai, user ko maintenance ki tension
nahi hoti.

•Self-managed deployment:
- User GitLab ko apne khud ke server par install aur manage karta hai.

•Built-in CI/CD:
- GitLab me built-in CI/CD, security tools, aur Kubernetes integration already available
hoti hai.
- It also support self-managed deployments.

•Security Features:
- GitLab advanced security features provide karta hai jaise ki vulnerability management,
container scanning, aur dependency scanning.

## GitLab  Vs  GitHub
| Feature            | GitLab                              | GitHub                                |
| ------------------ | ----------------------------------- | ------------------------------------- |
| Deployment Options | Cloud-hosted aur self-managed       | Mostly cloud-hosted                   |
| CI/CD Integration  | Built-in CI/CD tools                | Third-party tools ki zarurat hoti hai |
| Security Features  | Advanced aur comprehensive security | Basic security features               |
| Kubernetes Support | Native Kubernetes integration       | Limited Kubernetes support            |
| Pricing Model      | Free + paid plans                   | Free + paid plans                     |
