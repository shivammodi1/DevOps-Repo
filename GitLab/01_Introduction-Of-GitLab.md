# Introduction Of GitLab

- It is web-based DevOps platform.
- It provides a complete CI/CD toolchain.
- It supports both cloud-hosted and self-managed deployments.
```
# Important Features:
- Cloud-hosted deployment: GitLab is hosted and maintained by the service provider.
- Self-managed deployment: Users install and manage GitLab on their own servers.
```
- It provides built-in CI/CD , security and Kubernetes integration.
