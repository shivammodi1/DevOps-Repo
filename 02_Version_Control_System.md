# Version Control System (VCS)
- Track changes to code over time
- Collaborate with multiple developers
- Backup and Recovery
- Popular VCS tools: Git, SVN, Mercurial1`  

# Git (Most Popular VCS)
- It is a Popular Distributed Version Control System
- It is lightweight and fast Version Control System and it is free and open-source software.
- It will also provide Branching and Merging capabilities
- Speed and Efficiency is its key features

# Setup Git
- Install Git on your local machine
- git config --global user.name "Your Name" -> To set user information
- git config --global user.email "your.email@example.com" -> To set user information
- git config --list -> To verify configuration




# Version Control System Hosting Platform
- It is a centralized platform to host Git repositories
- Collaboration and Code Review features
- Backup and Deployment options
- Popular Hosting Platforms: GitHub, GitLab...

# Github 
- It is the most popular Git hosting platform
- Here We can collaborate with other developers
- We can contribute to open-source projects
- It provides features like Issues, Pull Requests, Actions for CI/CD
